Rules

The rules for the room are simple:

Before asking a question here, ask it on Stack Overflow or another Stack Exchange site.
Do not fire off a question before you've researched your problem. Remember, Google is always your friend!
Use English to the best of your ability and do not use chat-speak. Just try to make sense.
Pleasedon't bother ping spamming. This is the practice of constantly posting @username in order to annoy the person.
Try to limit in-chat code to a reasonable amount. If you have lots of code to share, how about making a gist, or even a repository. You can also use pastebin, pastie and hastebin. If you post code, it will be removed from chat.
Don't flag stuff unless it's obviously flag-worthy. It just annoys whoever invalidates the flags and doesn't help anyone.
Don't be creepy.

These are all non-strict and really we'll get along fine if you're not begging for code and help all the time. In other words, be excellent to each other.
